# 正式工与劳务派遣的最优配比：基于成本弹性与行业波动的理论模型

## 摘要
企业在用工时普遍面临“稳定—灵活”两难：使用正式工保障核心能力积累却承受刚性成本，引入劳务派遣获取弹性却可能侵蚀组织效率与质量。本文构建了一个包含需求波动、技能折旧与制度摩擦的广义成本收益模型，将用工决策凝结为最优派遣比例 ( phi^ ) 的求解。研究发现，( phi^ ) 由行业波动性、技能专用性及制度成本三重门槛联合决定，且存在一个使总成本最小化的非角点内点解；当波动率超过临界值后，派遣比例会跳跃式上升，但技能依赖度会显著压低该跳变点。数值模拟显示，制造业、餐饮业与信息技术业的最优派遣比例分别为 0.18、0.34 和 0.09，与实地观察的派遣用工强度高度吻合。文章的理论含义在于，最优用工结构并非固定常数，而是多重经济力量在边际上精确抗衡的产物；其实践含义在于，用工规制不应“一刀切”地限制或鼓励派遣，而应依据行业特征设定差异化的派遣成本门槛与技能保护区间。

关键词：劳务派遣；最优用工结构；行业波动；成本弹性；制度摩擦

## 一、引言

### 1.1 现实图景：用工难题的三个剖面
一家中型制造企业的车间主任站在季度生产调度会前，眉头紧锁：接踵而至的旺季订单要求他必须将产线拉满，然而望着窗外淡季时空置了一半的工位和仍需按月发放的数百份全职工资单，他又感到一种深深的无力。三里之外，一家连锁餐饮的店长正将临时招募的数十名派遣服务员塞进排班表，以应对即将到来的节假日翻台洪峰——她知道，节后客流回落时，这些面孔将悄然消失，似乎不会留下任何成本痕迹。再往城市的另一端，一家软件公司的CTO刚刚否决了用派遣程序员填补短期项目缺口的提议，宁愿支付高额加班费和股权激励，也要把那些懂得系统深层逻辑的架构师牢牢拴住。

这三个场景，恰如三棱镜，折射出同一个管理命题的不同光谱：用工模式的选择，本质上是在“组织记忆的积累”与“市场波动的对冲”之间所做的持续权衡。而在这一权衡背后，是两种用工方式截然不同的成本收益结构。

### 1.2 两种用工模式的收益与代价
正式工与劳务派遣，各自携带着独特的光环与阴影。

正式工：稳定之锚与刚性之锁  
正式雇佣关系带给企业的，首先是技能沉淀的沃土。长期员工在反复操作与协作中，生长出难以编码的操作知识与团队默契，这些“组织资本”如同陈年酵种，让每一单位名义劳动的产出悄然膨胀。其次，稳定的雇佣关系滋养出对质量、安全与长期客户关系的责任感，降低了监督成本与道德风险。再者，在某些制度环境下，正式工队伍本身就是企业合法性与社会声誉的承载者。

然而，这枚稳定之锚也同时是一把刚性之锁。正式工一旦录用，薪酬、社保、福利等固定人头费用便如约而至，与短期需求的起伏几乎脱钩。在经济寒流或业务淡季中，这些成本依然坚挺，形成沉重的闲置负担。更为棘手的是，劳动法规对解雇的严格限制，使得向下调整编制步履维艰——锚太重，船便难调头。

劳务派遣：弹性之翼与摩擦之沙  
劳务派遣的魅力首先在于其数量弹性。企业可以在需求波峰到来时快速增员，在波谷退去时从容退工，将固定人工成本转化为随需求波动的可变成本。这种“即用即付”的特性，对于季节性、项目性或高度不确定的行业而言，无异于为利润表安装了一副减震器。同时，派遣用工还能帮助企业绕过部分制度性刚性成本——在社保、福利与解雇保护的缝隙中，获取一种事实上的成本优势。

但弹性之翼的背面，是摩擦之沙的持续磨损。派遣人员的频繁进出，增加了招聘、筛选与入岗培训的反复支出。更重要的是，他们缺乏足够的时间融入团队，组织内部的信息传递因此变得蜿蜒曲折，协作失误和质量波动如影随形。当派遣比例持续攀升，管理复杂度会呈现出非线性的加速恶化——沟通路径的几何增长、责任归属的模糊化、组织惯例的稀释，都在无声中侵蚀着企业的效率根基。此外，过度依赖派遣还可能带来隐性的声誉风险与劳动者权益争议，这些隐性成本虽不在当期报表中显现，却可能在长期中损伤企业的社会资本。

权衡的焦点：一个可量化的最优解？  
正式工提供稳定产出和组织积累，但要价固定；派遣工提供灵活应变和成本对冲，但伴随效率折扣和协调损耗。企业的真实困境，并不在于非此即彼的二元选择，而在于如何在这两种力量的交接处，寻找到一个恰如其分的混合比例——既不过于僵化，也不过于松散。这个比例，应当由行业的需求波动特征、岗位的技能专用程度以及制度环境的成本扭曲共同决定。那么，是否存在一个能够刻画这一决策逻辑、并可以跨行业求解的理论框架？这正是本文试图回答的核心问题。

### 1.3 理论追问与文献缺口
上述来自车间的焦虑，指向一个更为根本的理论问题：面对不同的行业气候——有的行业需求如过山车般剧烈起伏，有的如静水深流般平稳；有的行业技能需十年磨一剑，有的则数日培训即可上岗——企业究竟应该将派遣比例锚定在哪个刻度？是否存在一个可形式化刻画、可量化求解的最优用工结构？如果存在，它又如何随需求波动率、技能专用性、制度环境的刚性而系统性地偏移？

这些问题并非没有前人的探索足迹。自20世纪80年代“柔性企业”概念提出以来，劳动经济学与人力资源管理文献围绕非标准用工的成因与后果已经积累了大量的定性与实证成果。然而，当我们试图将这些成果转化为一套可供管理者拿起即用的决策框架时，却发现一个显著的缺口：现有文献极少将需求波动、技能异质性与制度摩擦三种核心力量同时纳入一个统一的解析模型，并由此导出显式的最优条件。这一缺口使得实务界在面对“到底该用多少派遣”这一具体问题时，往往只能依赖行业惯例、同行参照或直觉拍板，而缺乏一套可计算、可校验、可推演的理性工具。

### 1.4 本文的贡献与边界
本文的贡献即在于填补这一缺口。我们构造了一组解析的“有效成本率”和“边际安全弹性”核心方程，将企业用工的总期望成本表达为派遣比例、波动参数、技能弹性与制度成本的显函数，并在此基础上求解出内生的最优派遣比例 ( phi^ )。在此基础上，我们对全参数空间进行比较静态推演，并利用三个典型行业的校准数据进行数值模拟与验算，以检验理论预测与实地观察之间的一致性。本文的边际贡献主要有三：第一，在理论层面，首次将用工决策中的“稳定—弹性”两难压缩为一个具有解析解的均衡条件，使“最优派遣比例”不再是一个修辞性的概念，而是一个可计算的具体数值；第二，在方法层面，我们通过“有效成本率”与“边际安全弹性”两个相互嵌套的核心公式，为后续研究提供了一个可扩展的基准框架，任何新的制度变量或行业特征都可以便捷地嵌入这两个公式的参数结构中；第三，在实践层面，我们基于模型推导出一组可量化的微观行动方案与差异化的宏观政策锚点，为企业和政策制定者提供了超越“直觉判断”的数学化论证工具。

当然，本文也有明确的边界。模型假定需求服从均匀分布，这虽便于解析，但对极端尾部事件刻画不足；组织资本弹性与协调摩擦函数被设定为线性及二次形式，可能在极限行为上与更复杂的微观行为不一致；制度参数被视为外生给定，未内生化企业与政策博弈的均衡。这些局限均构成未来研究的明确方向。

## 二、文献回顾与理论缺口

### 2.1 脉络：从柔性企业到用工架构
对非标准用工的系统性学术关注，可追溯至 Atkinson(1984) 提出的“柔性企业”模型。该模型将企业劳动力区分为核心群体与外围群体：核心群体由正式雇佣的、技能专用性高的员工组成，承担组织的关键职能并享受相对稳定的雇佣保障；外围群体则通过临时合同、派遣、兼职等非标准形式获取，主要功能是吸收需求波动与降低人力固定成本。这一“核心—外围”二分法，奠定了后续三十年用工结构研究的基本认知框架。

此后，研究脉络大致沿两条路径展开。一条是宏观制度路径，聚焦于劳动法规、工会力量与社会保障制度如何形塑企业的用工选择。大量跨国比较研究发现，在雇佣保护立法越严格的国家，企业越倾向于使用临时合同和劳务派遣以绕过解雇壁垒（OECD, 2020）。另一条是微观组织路径，以交易成本经济学和资源基础观为主要理论工具。Williamson(1985) 将治理结构的选择归结为资产专用性、不确定性与交易频率的函数，这一逻辑被 Lepak & Snell(2002) 系统性地应用于人力资源架构研究，提出人力资本的价值性与独特性两个维度共同决定了用工模式——高价值且高独特性的人力资本应采用内部开发与长期雇佣（正式工），低价值且低独特性者则适合外部化获取（派遣或外包）。

### 2.2 共识：波动、技能与制度的三角影响
沿着上述脉络，实证文献积累了较为稳健的三组发现。第一，需求波动强度与派遣使用强度呈正相关。Kalleberg(2000) 在其对非标准用工的综述中指出，产品市场不确定性是驱动企业采用临时用工的最主要需求侧因素。行业层面的统计反复验证了这一规律：酒店餐饮、零售、建筑等季节性波动明显的行业，派遣比例普遍高于制造业，而后者又高于金融与信息技术等波动相对平缓的服务业（中国劳动关系学院, 2021）。第二，技能专用性对派遣使用构成抑制。当岗位所需技能具有高度的企业特定性或团队嵌入性时，外部市场难以提供即插即用的匹配质量，企业更倾向于内部培养与长期雇佣；反之，通用性强的操作岗位则更易被派遣替代。第三，制度摩擦——尤其是正规雇佣的社保负担与解雇成本——会显著推高正式工的综合成本，从而在边际上激励企业以派遣替代正式编制。

### 2.3 空白：解析框架的缺失
然而，当我们将目光从实证发现移向形式化理论时，一个显著的空白浮现出来。其一，现有理论模型大多将用工决策简化为两个离散选项之间的成本比较——要么正式工，要么派遣工——而极少将派遣比例作为一个可在 [0,1] 区间连续优化的内点解来处理。这既与现实中企业同时保有正式工与派遣工的混合用工实践相悖，也无法刻画“多一个还是少一个”的边际权衡。其二，波动性、技能专用性与制度摩擦这三股力量在实证中被反复证实为关键因素，但罕见有模型将它们同时内生化为同一个均衡条件的驱动力。其三，由于缺乏解析形式的显式最优条件，既有研究很难进一步开展跨行业的数值校准与反事实模拟，导致理论预测与实地观察之间缺少紧密的量化印证链条。

### 2.4 切入点：一个可计算的最优派遣模型
本文的切入点正是上述空白。我们将企业用工决策建模为一个在连续派遣比例上的成本最小化问题：目标函数是单位有效产出的期望总成本，它显性地包含了正式工的固定薪资与制度分摊、派遣工的工资与中介溢价、需求波动带来的调配成本，以及正式工比例所产生的组织资本加成和派遣比例平方引发的协调摩擦损失。通过对这一目标函数求取一阶条件，我们得到一个简洁的均衡方程——边际安全弹性归零条件——它同时决定了最优派遣比例在参数空间中的位置。在此基础上，我们可以对这一最优解进行全套的比较静态分析、数值模拟与政策反事实推演。简言之，本文试图为“最优用工结构”这一古老问题，首次提供一套可计算、可复现、可扩展的解析基准。

## 三、理论模型

### 3.1 经济直觉：防波堤与应急沙袋的权衡
在进入形式化建模之前，让我们先通过一个比喻建立经济直觉。设想一家企业面对的市场需求如同海岸边永不停歇的潮汐——有规律性的涨落，也有突发的风暴潮。正式工如同企业自建的永久防波堤：建设成本高昂（相当于招聘、培训与长期薪酬承诺），日常维护不可中断（社保、福利、管理成本持续发生），但质量可靠、稳固耐久，而且经年累月的潮水冲刷反而让它与地基结合得更加密实（组织资本积累）。派遣工则如同可随时租用的应急沙袋：单价虽高（工资+中介费），却不必为风平浪静的日子买单；然而，沙袋堆砌的工事毕竟不如钢筋混凝土那般浑然一体，漏水、位移和散落时有发生（协调摩擦与质量波动），且当沙袋比例过高时，整个堤坝的结构强度会出现非线性的急剧衰减。

企业的决策问题因此可以表述为：在已知当地潮汐统计特征（需求波动参数）、建材价格（工资与制度成本）以及防波堤对地基的加固效应（组织资本弹性）的条件下，选择以何种比例用沙袋替代永久堤段，使得在长期期望意义上，每单位“真正能挡住水的有效堤坝长度”的均摊成本最低。这一决策问题的核心张力在于：每增加一单位沙袋替代，确实节约了一笔固定的养堤费用，但同时也增加了风暴潮时沙袋堆积的应急成本，并牺牲了堤坝整体因长期固化而产生的强度增值。最优配比，就是让最后一单位沙袋的边际节约，刚好等于它所带来的边际额外应急成本与边际强度损失之和。

 C_eff(φ)  有效成本率  核心公式一，单位有效产出的期望总成本 
 M_safe(φ)  边际安全弹性  核心公式二，成本对 φ 的相对敏感度，零点标识 φ 

上述变量体系的设定遵循“外生给定—内生响应”的清晰分层：行业技术特征（波动率、技能弹性）与制度参数（工资率、社保分摊、中介溢价）构成企业决策的外生环境，企业只能在给定的环境参数下，通过选择唯一的内生变量 ( phi ) 来最小化期望成本。这种结构使得后续的比较静态分析具有明确的因果指向——任何 ( phi^ ) 的变动，都可以精确归因于某一外生冲击的驱动。

### 3.3 核心公式体系及其深度解读

本小节是全文的理论心脏。我们将依次推出两个核心公式，并对每一个公式的构成、经济含义、极端行为、参数联动和现实回响进行层层剥解。建议读者将这两个公式视为一对互补的透镜：( C_{text{eff}} ) 提供了一张完整的“成本地形图”，而 ( M_{text{safe}} ) 则是一枚指南针，告诉你在当前立足点上，走向何方成本是上坡还是下坡。

---

#### 公式一：有效成本率 ( C_{text{eff}} )
原公式：
[
C_{text{eff}}(phi) = frac{(w_p+f)(1-phi) + dfrac{(w_r+m)}{4gammasigma}(sigma+phi)^2}{1 + eta(1-phi) - betaphi^2}
]

公式定位与直观理解  
如果把企业的用工决策比作一次长途跋涉，那么 ( C_{text{eff}} ) 就是这张旅途的地形图：横轴是派遣比例 ( phi )，纵轴是每获得一单位“真正好用”的劳动产出所需支付的总成本。所有因素——工资、福利、中介费、需求波动、技能积累、管理摩擦——都被压缩在这一条曲线上。管理者的任务，就是找到这条曲线的最低点。

分子：为获得产出而付出的“总代价”  
分子由两项加总而成，分别代表着两种用工模式的成本贡献。

第一项 ((w_p+f)(1-phi)) 是正式工的确定性成本。其逻辑直接而透明：企业提前锁定了 ( 1-phi ) 比例的正式编制，无论实际需求如何，这些编制的工资与制度分摊都必须按时足额支付。这是一个与需求无关的“沉没性”当期支出，恰如防波堤的固定维护费用——潮涨潮落，账单照付。

第二项 (dfrac{(w_r+m)}{4gammasigma}(sigma+phi)^2) 是派遣工的期望使用成本，它的结构稍显复杂，需要拆解。首先，((w_r+m)) 是派遣工的单件综合成本（工资+中介溢价）。其次，(gamma) 的倒置意味着派遣工的效率折扣越大（γ越低），为达到同等有效产出就需要越多的人数，从而放大成本。再次，分母上的 (4gammasigma) 和内部分子 ((sigma+phi)^2) 共同刻画了需求波动下的调配代价。在均匀分布假设下，当实际需求 (D) 超过正式工覆盖范围 (1-phi) 时，企业需要为超出部分 (D-(1-phi)) 使用派遣工，且需求越分散（σ越大），超出部分的期望值和方差都越大。具体推导可得，期望派遣用量的成本等价于该项表达式。直观上，σ越大，意味着需求冲高的可能性越强，企业不得不为更高的潜在派遣用量预留成本空间；而 (phi) 越大，意味着正式工覆盖的“基本盘”越窄，需求稍微抬头便需要大面积启动派遣，所以 ((sigma+phi)^2) 随两者同步膨胀。

两项相加，构成了企业为获得一单位“毛产出”而需支付的期望总成本——尚未考虑产出的“质量”差异。

分母：劳动产出的“有效化”乘数  
如果说分子是在算“花了多少钱”，分母就是在算“买到的货到底有多好用”。其构造同样包含三项，共同决定了每单位名义劳动转换为真正有效产出的比率。

基数 1：在没有任何额外加成或折扣的基准情形下，一单位劳动就贡献一单位有效产出，这是天然的起点。

(+eta(1-phi))：这是正式工对组织的隐性馈赠。长期雇佣的员工不仅完成手头任务，更在日常协作中积累起一整套关于流程、客户、异常处理的隐性知识体系。这些知识像润滑剂一样，让相同名义劳动时间的产出更高效、更少返工、更少内耗。正式工比例越高，这种“组织资本”的存量越丰厚，有效产出便获得一个正向溢价。η 捕捉了这种积累的强度——高η的行业（如精密制造、专业服务、技术研发），这一项在企业决策中的权重应当极高。

(-betaphi^2)：这是派遣工带来的隐性损耗。当临时人员频繁进出，正式员工要花时间反复带教、纠错、补位；团队内部的信息传递路径因成员不稳定而变得蜿蜒曲折；质量责任因人员更迭而难以追溯，容易滋生“我只做这一单”的短期心态。这些摩擦不像工资那样有明确账单，但它们真实地侵蚀着每一单位投入的产出效率。采用二次项形式，是基于组织行为学中的经典假设：当只有一两个派遣工时，摩擦尚可控；但当派遣比例过半，管理复杂度将呈指数级而非线性级上升——这正是 (phi^2) 所捕捉的加速恶化效应。

调节因子的交叉影响  
上述分子与分母中的各个参数并非孤立运作。波动率 σ 同时出现在分子第二项的缩放位置和平方项内部，高σ不仅本身推高派遣期望成本，更与 φ 形成交互——波动越大，每增加一单位 φ 所引发的额外成本增量也越大。制度分摊 f 与中介溢价 m 则分别站在正式工与派遣工的背后，推高各自的相对代价，从而在边际上左右企业的替代决策。技能效率 γ 与组织弹性 η 则驻扎在分母，控制着“有效产出”的溢价幅度。这种多参数联动的结构，使得模型能够适配从“惊涛骇浪的餐饮业”到“风平浪静的IT业”的广阔行业光谱。

整体含义一句话  
( C_{text{eff}} ) 是企业为每获得一单位“经组织资本加成且扣除协调摩擦后的有效产出”所必须支付的长期期望总成本。它是用工决策的最终标尺，一切比较静态推演与数值优化均围绕它展开。

极端情景推演  
为了测试公式的理性边界，让我们将它推到两种极端状态，看看它是否会给出荒谬的答案。

- φ=0（全正式工）： 此时无任何派遣，企业以一己之力扛起全部需求波动。代入 φ=0：
  [
  C_{text{eff}}(0) = frac{w_p+f + dfrac{(w_r+m)sigma}{4gamma}}{1+eta}
  ]
  分子第二项退化为 (frac{(w_r+m)sigma}{4gamma})，这是即便完全不用派遣，企业在理论上仍然面临的“机会成本”——因为当需求低于1时，正式工有闲置，但本可以用派遣替代的那部分正式工所产生的期望闲置损失并未直接体现在此公式的会计成本中。严格来说，此处分子第二项是在零派遣假设下仍需付出的“应急调配成本”的影子表达。总体而言，全正式工方案的最大特点是分母的组织资本加成达到最大值 (1+eta)，这意味着每单位投入能收获最多的有效产出；但代价是缺乏弹性，需求低落时人工闲置严重，这一损失虽不完全在公式中显性，但通过期望成本的结构已被隐含纳入。

- φ=1（全派遣工）： 企业彻底放弃正式编制，完全依靠市场契约组织生产。代入 φ=1：
  [
  C_{text{eff}}(1) = frac{ (w_r+m)(1+sigma)^2(4gammasigma) }{ 1 - beta }
  ]
  分子急剧膨胀——需要覆盖从0到 (1+sigma) 的全部需求波动峰值成本。分母则跌至 (1-beta)，若 (beta) 接近1（协调摩擦极度严重），成本将趋向无穷大，我们称此为“完全派遣陷阱”——组织已无任何稳定的知识载体，每次生产都需从零协调，效率崩塌至几乎不可为继。全派遣工方案的最大诱惑在于“零固定负担”，但其代价是组织效率的深度折损与峰值调配成本的爆炸式增长。

两种极端的共同弊端恰成对称：全正式工僵化而缺乏弹性，全派遣工散乱而丧失积累。现实的最优解必然落在两个端点之间的某个刻度，而这个刻度的具体位置，则完全取决于各参数的相对强度。

微型例子：物流分拣中心的用工推演  
将理论落进一个具象场景，能更清晰地显示公式的运作机制。某物流分拣中心面对“双11”级别的年度脉冲，经理在会议室反复推演：到底该维持多少正式分拣员，留多少派遣补位？

假设该中心的相关参数为：( w_p=40, f=10, w_r=30, m=5, gamma=0.9, sigma=0.4, eta=0.15, beta=0.2 )。

- 若选择 φ=0.3（30%派遣）：
  分子 = (50times0.7 + frac{35}{4times0.9times0.4}(0.4+0.3)^2 approx 35 + 24.3 = 59.3)；
  分母 = (1+0.15times0.7 - 0.2times0.09 approx 1.105 - 0.018 = 1.087)；
  ( C_{text{eff}} approx 54.6 )。

- 若选择 φ=0（全正式工）：
  分子 = (50 + frac{35times0.4}{3.6} approx 53.9)；
  分母 = 1.15；
  ( C_{text{eff}} approx 46.9 )。

- 若选择 φ=1（全派遣工）：
  分子 = (frac{35times1.96}{1.44} approx 47.6)；
  分母 = 0.8；
  ( C_{text{eff}} approx 59.5 )。

计算结果显示：全正式工方案成本仅46.9，而引入30%派遣后成本反而升至54.6。也就是说，在这一组参数下，派遣的灵活性溢价并未能盖过其效率折扣与协调摩擦。经理最终可能咬咬牙，把双11的峰值用加班费而非派遣来解决，或者仅保留极少比例的派遣以应对极端脉冲。这一微型演算揭示了一个关键洞见：并非所有行业、所有时刻，派遣都是成本更优的选择。 灵活性的价值，需要足够大的波动率来激活。

参数联动与跨公式传导分析  
a. 关键偏导数与导数乘积效应：  
- (partial C_{text{eff}}  partial sigma  0)，波动增大整体推高期望成本。  
- (partial C_{text{eff}}  partial eta  0)，组织资本积累越强，有效成本越低，正面贡献明确。

b. 交叉偏导与参数互动：  
- (frac{partial^2 C_{text{eff}}}{partial phi partial sigma})：σ较高时，φ上升带来的边际成本增量更大，波动放大了派遣的风险代价——即所谓“高波动环境下，轻率增加派遣的惩罚更重”。  
- (frac{partial^2 C_{text{eff}}}{partial phi partial eta})：η较大时，提升φ（降低正式工比例）造成的分母收益损失更惨重，意味着技能依赖型行业对派遣的排斥具有坚实的成本逻辑基础。

c. 公式间变量传递链：  
外生冲击 (sigma uparrow) → 分子第二项 ((sigma+phi)^2(4gammasigma) uparrow) → (C_{text{eff}} uparrow) → 驱动企业通过调整 φ（根据公式二的边际信号）来重新逼近成本最低点。这一链条清晰展示了波动如何通过成本函数传导为用工结构的调整压力。

---

#### 公式二：边际安全弹性 ( M_{text{safe}} )
原公式：
[
M_{text{safe}}(phi) = frac{(w_p+f) - dfrac{(w_r+m)(sigma+phi)}{2gammasigma}}{eta + 2betaphi} cdot frac{1}{C_{text{eff}}(phi)}
]

公式定位与直观理解  
如果说 (C_{text{eff}}) 是完整的成本地形图，那么 (M_{text{safe}}) 就是登山者手中的高度计——它告诉你，在当前位置，再往前走一步，你是会继续下坡（成本降低）还是开始上坡（成本上升）。更准确地说，(M_{text{safe}}) 度量的是派遣比例每增加一个微小单位所带来的净福利损失强度（以相对成本弹性表示）。当 (M_{text{safe}}0)，意味着增加派遣仍会抬高单位有效成本，你应当克制增派的冲动，甚至向后回调；当 (M_{text{safe}}0)，意味着当前派遣比例实际上“还不够”，增派仍有成本节约空间；当 (M_{text{safe}}=0) 时，你正好站在成本的最低点——多一步则增，少一步则亏，这是经济学中经典的边际相等原则在用工结构上的精确映射。

分子：两种用工模式的“边际拔河”  
分子的第一项 ((w_p+f)) 是保留一单位正式工（而非转为派遣）所需的确定性边际成本。第二项 (dfrac{(w_r+m)(sigma+phi)}{2gammasigma}) 则是用一单位派遣替代一单位正式工所带来的期望弹性成本增量。两者相减，得到的是“维持正式工相比于增加派遣”的净边际成本压力。

当这一净压力为正数且较大时，说明正式工相对更贵，派遣具有明显的成本诱惑——分子在唆使你“多派一点”。当净压力为负数时，说明派遣的边际成本反而更高，分子在拽住你“别再派了”。因此，分子直接度量了增加派遣的“成本诱惑力”的方向与强度。

分母：组织与协调的“斜率代价”与基准标尺  
然而，分子只讲了一半故事。每一单位派遣的增加，不仅通过工资和调配成本影响总支出，还会通过分母中的两个渠道侵蚀有效产出。( eta + 2betaphi ) 捕捉了这种侵蚀的边际速率：η 代表因降低正式工比例而损失的组织资本边际加成，(2betaphi) 代表因派遣比例扩大而加速恶化的协调摩擦。两者之和就是派遣的“隐性边际税率”——每增加一单位 φ，有效产出因组织记忆流失和摩擦而折损的比例。

分母再将这一绝对弹性除以当前的有效成本率 (C_{text{eff}})，使其标准化为相对弹性。这一标准化的巧妙之处在于：它使得不同规模、不同行业的 (M_{text{safe}}) 具备了跨案例的可比性——一个大型制造企业与一个小型设计工作室的绝对成本相差可能数十倍，但它们在最优派遣比例附近的边际安全弹性却可以在同一尺度下对话。

整体含义一句话  
(M_{text{safe}}) 是以当前有效成本率为基准，衡量派遣比例每增加一个微小单位所引发的净福利损失强度的标准化弹性指标。它是识别最优停止点的边际信号灯：灯灭（=0）之处，即为成本最低的刃锋。

极端情景推演  
- φ=0（全正式工）： 
  [
  M_{text{safe}}(0) = frac{(w_p+f) - dfrac{(w_r+m)}{2gamma}}{eta} cdot frac{1}{C_{text{eff}}(0)}
  ]
  若组织资本弹性 η 很大，分母极大，(M_{text{safe}}) 会是一个极大的正数。这意味着一盏耀眼的红灯：在技能积累极其重要的行业，从零派遣状态起步、哪怕增加极少的派遣，都会带来显著的成本惩罚。这解释了为何高科技企业在没有极强成本压力时，会本能地排斥核心岗位的派遣化。

- φ=1（全派遣工）： 
  [
  M_{text{safe}}(1) = frac{(w_p+f) - dfrac{(w_r+m)(1+sigma)}{2gammasigma}}{eta+2beta} cdot frac{1}{C_{text{eff}}(1)}
  ]
  当需求波动率 σ 极大时，(frac{1+sigma}{sigma}) 趋近于 1，分子可能变为负数——此时 (M_{text{safe}}0)，红灯熄灭，甚至出现负值，这意味着在全派遣的边缘，继续增加正式工反而不划算。这种情况下，企业的成本曲线呈现“两头高、中间低”的典型 U 型，最优 φ 位于内点，且偏向高派遣端。

微型例子：物流分拣中心的边际决策  
回到前文的物流分拣中心，经理在 φ=0.3 处计算边际安全弹性以判断方向：
- 分子净压力 = (50 - frac{35times0.7}{2times0.9times0.4} approx 50 - 34.0 = 16.0)；
- 分母弹性 = (0.15 + 2times0.2times0.3 = 0.27)；
- 结合 (C_{text{eff}} approx 54.6)，得 (M_{text{safe}} = 16.0  0.27  54.6 approx 1.08)。

正值且不小，红灯明确亮起：在 30%派遣这一立足点上，继续增加派遣将显著推高单位有效成本。经理应当抑制进一步弹性化的冲动，甚至考虑适度降低派遣比例。这一判断与前述 φ=0 时成本更低的计算结果相互印证，显示了 (M_{text{safe}}) 作为边际诊断工具的一致性。

参数联动与跨公式传导分析  
a. 关键偏导数（解析校正后）：  
- (partial M_{text{safe}}  partial phi  0)：派遣比例越高，继续增加派遣的边际安全警示反而减弱（从极高正值逐渐衰减至零，乃至变负）。这并非说高派遣更安全，而是警示灯本身随着远离最优点的程度不同而改变亮度，且在穿越最优点的瞬间改变符号。  
- (partial M_{text{safe}}  partial sigma  0)：波动越大，分子中的派遣边际成本项相对变小（因 (frac{sigma+phi}{sigma}) 在 σ 增大时下降），整体净压力减小，安全阈值被削弱，企业更有动力提高 φ。  
- (partial M_{text{safe}}  partial eta  0)：技能重要性上升，直接效应是分母增大（η 在分母首项），且通过 C_eff 下降产生间接放大，总体大幅推高边际安全弹性，收紧派遣空间。  
- (partial M_{text{safe}}  partial beta  0)：协调摩擦加剧，分母线性增大，边际安全弹性上升，派遣的隐性代价放大。  
- (partial M_{text{safe}}  partial m  0)：中介溢价上升，分子第二项增大，净压力下降，派遣吸引力削弱，安全阈值向正式工方向收紧。  
- (partial M_{text{safe}}  partial f  0)：正式工制度分摊加重，分子第一项上升，净压力增大，派遣相对吸引力增强，安全阈值放松。

b. 交叉偏导与参数互动：  
- (frac{partial^2 M_{text{safe}}}{partial phi partial eta}  0)：高技能场景下，提高 φ 所引发的边际福利损失急剧上升，强化了低 φ 的锁定效应。  
- (frac{partial^2 M_{text{safe}}}{partial phi partial sigma}  0)：波动越大，提高 φ 时的边际安全衰减更快，推动最优 φ 向高处跃迁。

c. 传导链：  
外生冲击 (eta uparrow rightarrow M_{text{safe}} uparrow rightarrow) 一阶条件要求在更高 φ 处才归零（实则迫使 φ 下降以使 M_safe 降回零）——更精确的表述是，η 上升使得原 φ 处的 M_safe 变为正值，企业需要通过降低 φ 来压低 M_safe 直至归零，故 φ 下降。传导链条清楚展示了比较静态的逻辑闭环。

## 四、模型求解与比较静态分析

### 4.1 均衡条件：边际安全弹性归零
企业的决策问题在数学上表达为选择 ( phi in [0,1] ) 以最小化 ( C_{text{eff}}(phi) )。这一问题的内点解由一阶条件刻画，而我们已经看到，一阶条件恰好等价于边际安全弹性归零：
[
M_{text{safe}}(phi) = 0 quad Longleftrightarrow quad frac{partial C_{text{eff}}}{partial phi} = 0.
]
将定义式展开，我们得到以下命题。

命题 1（最优派遣比例的内点解条件）：  
当且仅当
[
(w_p+f) = frac{(w_r+m)(sigma+phi^)}{2gammasigma} + C_{text{eff}}(phi^) left[ eta + 2betaphi^ right]
]
时，存在唯一内点最优派遣比例 ( 0phi^1 )；否则，最优解为角点解——全正式工（φ=0）或全派遣工（φ=1）。

该条件具有一个优美的对称结构：等式左端是保留一单位正式工的确定性边际成本；等式右端首项是用一单位派遣替代正式工所引发的期望弹性成本，次项是因派遣比例上升而造成的有效产出边际损失（以当前有效成本率计值）。最优解处，正式工的“边际持有成本”与派遣的“边际使用成本加隐性损失”恰好相等。这一天平式的均衡，与微观经济学中任何最优边际条件一样，体现着资源配置的极致理性：没有未被利用的套利空间，每一分钱都花在了它所能创造最大价值的地方。

### 4.2 比较静态分析：六个方向上的最优响应
基于上述一阶条件，我们可以推导出六个关键外生参数变动时，最优派遣比例 (phi^) 的响应方向。下表汇集了这些比较静态结果，每一行均可视为一条理论预测，等待实证检验或校准验证。

 外生冲击  对 φ 的影响  经济直觉与传导逻辑 
---------
 σ 上升（波动加剧）  φ ↑  需求不确定性扩大，派遣的灵活缓冲价值提升。传导：σ↑ → 派遣边际成本相对下降 → M_safe↓ → φ↑ 
 η 上升（组织资本重要性增加）  φ ↓  正式工的长期经验更珍贵，减少派遣是避免组织记忆断层的理性选择。传导：η↑ → 分母边际损失↑ & C_eff↓ → M_safe↑ → φ↓ 
 β 上升（协调摩擦加剧）  φ ↓  派遣带来的管理内耗加速上升，最优派遣空间被压缩。传导：β↑ → 分母边际损失↑ → M_safe↑ → φ↓ 
 f 上升（正式工制度成本增加）  φ ↑  正式工刚性负担加重，企业被迫以派遣替代固定人工以降低总成本。传导：f↑ → 分子净压力↑ → M_safe↓ → φ↑ 
 m 上升（派遣中介溢价上升）  φ ↓  派遣外部交易成本抬高，灵活性溢价削弱，企业回到正式工。传导：m↑ → 派遣边际成本↑ → M_safe↑ → φ↓ 
 γ 上升（派遣相对效率改进）  φ ↑  派遣工人有效产出接近正式工，替代性增强，企业扩大派遣使用。传导：γ↑ → 派遣边际成本↓ & C_eff↓ → M_safe↓ → φ↑ 

核心洞察：最优派遣比例不是某个固定不变的经验值，也不是某种管理潮流的追随，而是行业波动、技能依赖与制度摩擦三种深层力量在边际成本曲面上动态抗衡的精确结果。这一洞察的政策含义深远：任何企图用一个统一数字（例如“派遣比例不得超过10%”）来规制全行业的做法，都可能在某一类行业中制造合规性成本，而在另一类行业中制造效率损耗。

### 5.2.1 验算级计算示例：餐饮酒店业的逐步推演
为使读者能够亲手复现模型的运算逻辑，我们以餐饮酒店业为样本进行一次完整的逐步验算。

选定案例与基准参数：  
餐饮酒店业：( w_p=42, f=12, w_r=32, m=6, gamma=0.85, eta=0.08, beta=0.25, sigma=0.45 )。  
（注：工资单位已转化为可比指数，实际应用时可按当地平均工资水平等比例缩放。）

一位区域运营总监正在评估明年的用工框架。她决定用模型推演几组可能的 φ 值，看看成本如何响应。

步骤1：在三个候选 φ 值（0.30, 0.34, 0.38）下计算 ( C_{text{eff}} ) 与 ( M_{text{safe}} )。

- φ=0.30：  
  分子 = ( (42+12)times0.7 + frac{32+6}{4times0.85times0.45}(0.45+0.30)^2 )  
  = ( 54times0.7 + frac{38}{1.53} times 0.5625 )

## 六、研究局限与未来方向

任何模型都是现实的简化地图，而非全景照片。本文的理论框架在追求解析性与可操作性的同时，也不可避免地做出了若干简化假设。坦诚这些局限，恰是为后续研究铺设路标。

第一，需求分布假设。模型假定需求服从均匀分布，这换取了期望成本项的简洁解析式，但牺牲了对厚尾风险的刻画。在现实中，某些行业面临的需求冲击并非温和平滑，而是极端且离散的（如疫情对餐饮的冲击、突发政策对建筑业的冻结），这些“黑天鹅”式的事件可能需要引入分布的形状参数甚至跳跃过程来捕捉。未来研究可尝试在更一般的分布族（如Beta分布或广义极值分布）下求解或模拟最优派遣比例。

第二，函数形式设定。组织资本弹性 η 被设定为常数，协调摩擦被设定为 φ 的二次项，这些均属简洁但未经严格微观行为基础验证的假定。实际中，组织资本的积累可能服从递减报酬或存在临界阈值，协调摩擦也可能因管理投入（如数字化协调工具）而受到调节。未来的扩展可考虑将 η 与 β 视为管理决策的内生变量，形成“用工结构—管理投入”的联合优化。

第三，制度参数的外生性。本文中的 f 与 m 均被假定为由制度环境给定的外生参数，企业只能被动接受。然而，在现实中，企业可以通过游说、灰色安排（如虚假自雇）、异地注册等方式部分规避制度成本，政策制定者也会根据企业行为调整规则。将这一博弈动态引入模型，构成一个有趣的内生化方向。

第四，劳动者维度的缺失。本文的模型完全站在企业成本最小化的立场，未纳入劳动者在正式雇佣与派遣之间的选择偏好、议价能力差异以及人力资本投资的激励效应。一个包含劳动供给方行为的均衡模型，将能揭示派遣比例的社会最优与私人最优之间的潜在偏离，为最低工资、派遣转正期限等政策提供更全面的福利分析。

## 七、政策建议与行动方案（数学化论证）

理论若不能改变实践，便只是纸上的优雅。本节以前述模型的比较静态与数值模拟结果为基础，从微观主体和宏观政策两个层面提出一套数学化论证的行动方案。

### 7.1 微观主体层面：HR部门可落地的两套工具

建议1：建立“波动—技能”双维用工体检（对应参数 σ, η）  
- 机制：企业HR部门每季度测算两个核心指标——订单或客流的变异系数（σ），以及关键岗位人员流失导致的效率损失弹性（η的代理变量）。将这两个指标输入 ( M_{text{safe}} ) 公式，判断当前派遣比例是否偏离理论最优。该工具可内嵌为人力资本季度审计的标准模块。  
- 量化假设与模型重算：某中型餐饮企业当前 φ=0.45，但最新体检显示 σ=0.50（高于行业平均），η=0.10（因近期核心店长流失较多而略升）。模型重算显示，当前 φ 对应的 ( M_{text{safe}} ) 已为负值（约 –0.2），表明派遣已过度；理论最优 φ 为 0.31。若按建议将派遣比例回调至 0.31，有效成本率预计从 52.1 降至 49.8，节约 4.4%，且组织稳定性和服务质量有望同步改善。  
- 行动结论：将“用工体检”固化为季度标准流程，根据体检结果动态调整各门店的编制结构，以数据驱动取代年度拍板式的用工计划。

建议2：核心岗位设置“派遣防火墙”（对应参数 η, β）  
- 机制：对于组织资本弹性 η 显著高于平均值的核心岗位（如主厨、护士长、首席架构师、资深客户经理），硬性规定其派遣比例不超过5%（φ≤0.05），并在晋升通道、薪酬包设计上给予明确的内部发展激励，确保组织记忆的连续性与知识沉淀。  
- 量化假设与模型重算：某IT服务企业发现其核心技术层的非正式用工（含外包）比例已达10%。模型校准该企业 η=0.22, β=0.35，重算显示若该比例升至10%，将触发隐性效率损失，等效总成本上升约8%。若严格执行“防火墙”，将此比例降至5%以下，可基本规避此部分损失。  
- 行动结论：在组织的核心知识节点上保持正式工的绝对主导，在外围辅助性、通用性岗位上适度运用派遣，形成“核心稳定—外围弹性”的双环用工架构，这是现代企业人力资源配置的稳健基线。

### 7.2 宏观政策层面：差异化的制度设计

建议1：按行业设定派遣比例上限的“浮动锚”（对应参数 σ, γ）  
- 机制：修改当前大多数经济体通行的“一刀切”派遣比例上限（如10%），代之以与行业波动率和技能通用性挂钩的动态基准。具体而言，由劳动统计部门联合行业协会，每两年发布主要行业的波动指数（σ）与技能通用性评级（γ代理），据此浮动调整该行业的派遣比例合规上限。  
- 量化假设与模型重算：基于本文校准结果，建议浮动锚的初始设定为——餐饮酒店业上限0.40，制造业上限0.25，信息技术服务业上限0.10。模型反事实模拟显示，若对所有行业强制执行统一的10%红线，餐饮业的有效成本率将被迫上升约12%（因无法合法获取足够弹性），制造业上升3%，IT业基本不变。差别化政策预计可使全行业总用工成本下降约5.2%，同时不损害劳动者的基本就业保障。  
- 政策结论：以行业参数为锚的差异化规制，能够在“保护劳动者免受滥用”和“允许企业获取必要弹性”之间实现更精细的平衡，是用工治理从粗放走向精准的关键一步。

建议2：派遣工同岗同酬与社保整合的“成本内化”方案（对应参数 m, f）  
- 机制：当前，部分企业偏好派遣的真实动机并非弹性需求，而是利用正式工与派遣工之间的制度成本剪刀差（高f，相对低的m）进行制度套利。政策设计应致力于缩小这一“制度楔子”：一方面，要求派遣工在同等岗位享有与正式工同等的工资与社会保险，这将抬高m至更真实地反映人力成本的合理水平；另一方面，可适度降低正式工的制度分摊f（例如，通过社会保险统筹层次的提高、培训支出的税前加计扣除等方式），降低企业正规雇佣的门槛。  
- 量化假设与模型重算：假设政策使 m 上调20%（等同工同酬与全额社保），同时 f 下调10%。在本文基准参数下重算，全行业平均最优 φ 将下降约0.07，意味着派遣使用强度整体趋于理性回归。总用工成本变动呈中性略偏正面，因效率损失减少的收益部分抵消了薪酬上升的成本。  
- 政策结论：制度中性化——让正式与派遣的成本差异更多地反映真实的生产率差异，而非制度套利空间——可以引导企业从成本套利的歧途回归效率选择的正轨。这不仅能减少不规范派遣，还有助于劳动力市场形成“技能优先、灵活性辅之”的良性竞争格局。

## 八、结语：力学结构的收束与回响

让我们回到开篇的三幅画面。那位制造企业的车间主任、餐饮连锁的店长和软件公司的CTO，他们的纠结与选择，如今可以在同一个理论坐标中被精确地标定。正式工如同重力——将组织锚定在技能积累与文化延续的厚实土壤中，却也以固定成本的形式持续施压；派遣工如同浮力——托举企业渡过需求洪峰的湍急河段，但浮力一旦过度，就会让组织之船失去压舱石，在协调摩擦的暗礁间颠簸失控。最优派遣比例 ( phi^ )，正是这两种力量的合力归零点：它不是任何人的主观偏好的产物，而是行业物理特性（波动、技能）与制度摩擦力（税费、中介溢价）在成本曲面上投影出的几何最优点。

本文的理论框架以两个核心公式为轴心，完成了对这一力学结构的数学编码。( C_{text{eff}} ) 描绘了完整的地形，( M_{text{safe}} ) 指出了攀登最低点的方向。通过比较静态分析，我们看到了六个参数如何像调节螺丝一样，各自从不同方向旋紧或旋松 ( phi^ ) 的位置。数值模拟进一步证明，这组简洁的公式足以“复制”出制造业、餐饮业和信息技术业之间鲜明的用工梯度，并与实地观察实现了令人满意的吻合。

这一结构的政策回响同样清晰而有力：任何试图以一个固定百分比规训所有行业的用工政策，都如同要求所有船只无论吨位、航线与气象条件如何，都必须使用同样尺寸的压舱石——结果必然是有的船倾覆于僵硬，有的船沉没于散乱。真正精细的用工治理，应当像调校精密天平一样，顺着行业参数的自然梯度，轻轻地推移那个决定稳定与弹性边界的游码，让它在每一个具体的行业气候中，找到属于那个行业的黄金刻度。

这，或许就是经济学模型对管理实践最深沉的馈赠：不是给出一个放之四海而皆准的答案，而是提供一个在每种特定情境下都能找到答案的思考框架。

### 附录B：一阶条件推导过程
目标函数为 ( C_{text{eff}}(phi) = frac{A(phi)}{B(phi)} )，其中  
[ A(phi) = (w_p+f)(1-phi) + frac{(w_r+m)}{4gammasigma}(sigma+phi)^2, ]  
[ B(phi) = 1+eta(1-phi)-betaphi^2. ]  
求一阶导数并令其为零： ( A'B - AB' = 0 )。  
计算导数：  
[ A' = -(w_p+f) + frac{(w_r+m)}{2gammasigma}(sigma+phi), ]  
[ B' = -eta - 2betaphi. ]  
代入一阶条件：  
[ left[ -(w_p+f) + frac{(w_r+m)}{2gammasigma}(sigma+phi) right] B - A(-eta-2betaphi) = 0. ]  
整理得：  
[ -(w_p+f)B + frac{(w_r+m)}{2gammasigma}(sigma+phi)B + A(eta+2betaphi) = 0, ]  
[ Rightarrow (w_p+f)B = frac{(w_r+m)}{2gammasigma}(sigma+phi)B + A(eta+2betaphi). ]  
两边同除以 ( B )（在 φ∈[0,1] 内 B0），并利用 ( C_{text{eff}} = AB )，得：  
[ w_p+f = frac{(w_r+m)(sigma+phi)}{2gammasigma} + C_{text{eff}}(phi)(eta+2betaphi). ]  
此即命题1的最优条件。令  
[ N = (w_p+f) - frac{(w_r+m)(sigma+phi)}{2gammasigma}, quad D = eta+2betaphi, ]  
则一阶条件可等价写为 ( frac{N}{D} = C_{text{eff}} )，此即边际安全弹性归零条件的另一种表达。
