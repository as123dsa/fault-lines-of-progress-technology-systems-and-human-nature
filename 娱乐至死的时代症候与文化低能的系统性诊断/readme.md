# PPT自动配音+自动播放操作手册
## 一、前置准备
1. 新建一个**独立文件夹**（建议英文路径，如`D:\PPT_Auto`），将**待处理的.pptx文件**放入该文件夹
2. 把`pptx_inject_voice.ps1`脚本和`auto_play.txt`文件，也复制到这个文件夹中
3. 确保电脑已安装PowerPoint（2016及以上版本）

---

## 二、第一步：PPT配音处理（执行PowerShell脚本）
### 操作步骤
1. 打开刚才存放PPT和脚本的文件夹
2. **在文件夹空白处**，按住`Shift`键 + 鼠标右键，选择【在此处打开PowerShell窗口】
3. 在弹出的蓝色PowerShell窗口中，输入以下命令并按回车：
```powershell
.\pptx_inject_voice.ps1
```
4. 等待脚本执行完成（窗口无报错、自动执行完毕），此时PPT已完成配音注入

---

## 三、第二步：添加自动播放VBA代码
### 1. 打开处理后的PPT文件
直接双击文件夹中**已配音的PPT**，打开PowerPoint软件

### 2. 打开VBA编辑器
按下快捷键：`Alt + F11`

### 3. 插入代码模块
1. 在左侧窗口找到：`VBAProject (你的PPT文件名.pptx)`
2. 右键点击该项目 → 选择【插入】→ 点击【模块】

### 4. 粘贴自动播放代码
1. 打开文件夹中的`auto_play.txt`文本文件，**全选复制里面所有代码**
2. 回到VBA编辑器的空白模块窗口，**粘贴复制的代码**

### 5. 运行代码实现自动播放
1. 点击VBA编辑器顶部菜单栏：【运行】→ 【运行子过程/用户窗体】
2. 或直接按快捷键`F5`，PPT即可自动播放+自动配音

---

## 四、常见问题说明
1. **脚本无法运行**：PowerShell权限不足，以**管理员身份**打开PowerShell，执行命令后再重试
2. **VBA代码报错**：检查PPT文件名是否为英文，不要包含特殊符号
3. **无声音**：确认脚本执行成功，电脑音量已打开

---

### 总结
1. 统一存放文件 → 运行PowerShell脚本完成配音
2. 打开PPT → `Alt+F11`打开VBA → 插入模块 → 粘贴`auto_play.txt`代码
3. 按`F5`运行代码，实现PPT自动配音+自动播放
