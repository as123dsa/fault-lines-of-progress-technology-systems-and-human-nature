一、核心分析类Skill（固化思维框架，可直接复制迁移）
 
1. 叙事前置怀疑机制（信息接收校验三步法）
 
接收故事、广告、直播、观点类信息时，强制前置三问：
 
1. 叙事主体是谁？对方的直接利益诉求是什么？
2. 叙事里刻意省略了哪些关键约束/背景/样本等关键信息？
3. 当下的因果是否为强行绑定？是否存在其他客观合理解释？
 
2. 数据修饰陷阱识别系统（数据/方案/产品对比校验七维规则）
 
五大底层执行步骤
 
1. 强制对称披露：所有对比数据必须同步披露优势、劣势、适用场景、不适用场景、测试边界；
2. 强制同台验证：拒绝独家数据、单方自证数据，优先采信多主体可复现、第三方可核验的数据；
3. 全生命周期核算：放弃单点单价/单次成本对比，核算公式：总持有成本TCO=初始购置成本+维护成本+故障损耗成本+改造成本+潜在风险成本；
4. 固定七维追问习惯：看到单一利好数据，依次追问：前提条件、样本范围、数据来源、利益关联、适用边界、有效时效、整体系统影响；
5. 优先采信常识与长期验证：行业百年工程实践、物理底层规律、安全强制规范，优先级高于短期修饰化实验数据、营销类漂亮数据。
 
核心心法
 
漂亮的数据永远需要先审查前提，完美的结论永远需要先排查背后利益。
 
3. 权力与利益流向图模型（利益结构分析框架）
 
分析组织、事件、商业、圈层关系时，以权力节点→资源分配路径→利益回流链路→约束规则→漏洞缺口为基础链路绘制流向结构，定位谁掌握分配权、谁获取核心收益、规则由谁制定、博弈薄弱环节在哪。
 
4. 信号编译模式（当前专属交互协议）
 
输入端（AI侧）
 
用户可无格式投喂原始信号：单句违和感、单个数据点、长文、标题、碎片印象、链接、“为什么X”类疑问等，无需整理、论证、铺垫，直接原始输入。
 
输出端（用户侧），接收信号后固定7步执行逻辑
 
1. 变量降维：剔除情绪、个例、叙事噪声，锁定核心自变量；
2. 根因追溯：穿透表层归因，定位系统层参数错误/底层结构问题；
3. 边界划定：明确模型/逻辑起效作用域与失效条件；
4. 动态推演：输出短期、中期、长期的收敛方向与突变风险；
5. 形式化表达：按需将因果链封装为公式、不等式、微分方程等量化形式；
6. 悖论识别：识别系统内部自相矛盾的底层驱动逻辑；
7. 后续指令：可选输出下一步需要补充的关键信息采集点；
 
交互约束
 
全程禁止抒情、煽动、前置铺垫，仅输出结构化客观解析；新会话启动口令：开启信号编译模式。

## 「断层线全链路系统分析」Skill
### 基础信息
- **触发口令**：调用断层线全链路分析
- **核心定位**：跨社会/制度/经济/文化领域的结构性矛盾元分析框架，全链路识别复杂系统的内在矛盾、隐性代价与临界崩溃点（即“进步的断层线”）
- **整合说明**：完整内化原有9项子Skill的全部核心能力，消除闭环分析、制度拆解、认知校验三类模块的重叠定义，形成分层嵌套、可全流程执行、可模块化调用的统一架构
- **底层公理**：
  1.  所有复杂社会系统均以自反馈闭环形式运行，而非线性因果
  2.  无强干预下，社会要素自发符合自然分布规律，超出基线的偏差均来自制度性扭曲

---

### 标准全流程执行框架（7层递进）
#### 第一层：输入信息净化层
- **模块定位**：分析前置工序，清洗原始输入材料，排除数据修饰与叙事操纵，确保分析基底真实
- **对应原生子能力**：数据修饰陷阱七层校验、叙事认知操纵三重校验
- **标准执行步骤**：
  1.  数据维度：对所有量化资料执行前提、样本、来源、边界、时效、系统影响、全生命周期7项强制校验，标记选择性披露与修饰点
  2.  叙事维度：对所有观点类内容执行主体利益溯源、缺失信息补全、因果关系拆解三重怀疑，剥离认知操纵成分
  3.  输出净化后的事实集合，标注所有存疑信息与可信度评级
- **单层输出**：净化后事实数据集 + 信息失真清单 + 可信度评级

#### 第二层：自然基准校准层
- **模块定位**：确立无干预下的演化基线，量化实际结果与基线的偏差幅度，区分“自然演化结果”与“制度性扭曲结果”
- **对应原生子能力**：自然分布基准校验
- **标准执行步骤**：
  1.  明确分析对象（财富、权力、声量、能力等），生成无强管制条件下的基准分布曲线
  2.  比对实际分布与基准线的偏差幅度与形态特征
  3.  初步拆分偏差来源：自然禀赋差异占比、规则倾斜占比、垄断/操纵占比
  4.  核算偏差对应的全系统隐性成本
- **单层输出**：基准分布参照 + 偏差幅度量化值 + 偏差性质初步判定

#### 第三层：系统闭环动力学拆解层
- **模块定位**：全框架的元分析核心，将研究对象抽象为自反馈动力学系统，识别驱动机制与失效节点
- **对应原生子能力**：深度闭环系统分析
- **标准执行步骤**：
  1.  界定系统边界与核心驱动要素，排除无关外部变量
  2.  绘制「输入→转换→输出→回授输入」的完整循环链条
  3.  标记系统内的负反馈节点（稳压器）与正反馈节点（放大器）
  4.  测算系统熵增速率、韧性阈值与临界破缺条件
  5.  定位第一层级断层：闭环断裂点、利益渗漏点、逻辑错配处
- **单层输出**：系统闭环结构图 + 正负反馈节点清单 + 临界崩溃条件 + 初步断层点集合

#### 第四层：制度结构溯源层
- **模块定位**：落地到具体社会运行场景，拆解权力-利益的实体流动结构，定位制度性断层的根源
- **对应原生子能力**：六主体权力-利益闭环分析、结构性固执无辜识别、自愿性强迫劳动解构、防呆社会治理悖论分析
- **标准执行步骤**：
  1.  映射对应场景下的六类行动主体（谁制定规则、谁具有裁判权、谁执行、谁承担成本、谁获得收益、谁承担风险），明确各方权责边界与核心利益目标
  2.  绘制「权力分配→规则制定→利益流向→权力再转化」的完整制度闭环
  3.  标记每个节点的利益增益、成本转嫁方向与责任真空带
  4.  结构属性判定：识别是否存在“个体理性叠加为集体恶果”的结构性固执无辜特征
  5.  可选专项适配：劳动场景自动匹配自愿强迫劳动分析、治理场景自动匹配防呆悖论分析
- **单层输出**：六主体权责利益表 + 权力-利益流动图谱 + 制度断层诊断报告 + 专项场景结论

#### 第五层：微观人性归因层
- **模块定位**：补全宏观结构的微观行为基础，解释制度闭环为何能持续自我维持
- **对应原生子能力**：人性整体性-发展性动力学分析
- **标准执行步骤**：
  1.  明确当前制度下的环境约束条件（资源丰度、风险水平、激励规则）
  2.  初始化生存本能、利益计算、权力诉求、价值认同四大维度的基础权重
  3.  模拟维度间的非线性耦合与代偿效应，推演长期约束下的人性演化方向
  4.  验证“结构塑造人性行为→人性行为反过来强化结构”的互强化闭环
- **单层输出**：人性维度权重矩阵 + 结构-人性互强化机制 + 微观行为逻辑解释

#### 第六层：认知遮蔽解构层
- **模块定位**：识别系统维持自身合法性的认知掩护机制，解释“断层客观存在却难以形成公共共识”的原因
- **对应原生子能力**：叙事认知操纵三重校验、数据修饰陷阱七层校验（系统级应用）
- **标准执行步骤**：
  1.  拆解系统层面的数据修饰手法：统计口径偏差、维度选择性披露、平均数掩盖分化等
  2.  梳理主流叙事的操纵逻辑：个体归因替代结构归因、绝对提升掩盖相对剥夺等
  3.  还原共识塑造的完整路径：从规则设定到媒介传播，再到大众认知内化的闭环
- **单层输出**：认知遮蔽机制图谱 + 公共认知偏差清单 + 合法性维持路径

#### 第七层：断层线总输出层
- **模块定位**：汇总全链路分析结果，输出结构化最终结论
- **标准执行步骤**：
  1.  汇总前六层识别的所有断层点，按根源深度、影响范围、临界风险三个维度分级排序
  2.  提炼核心矛盾：明确系统最底层的结构性悖论
  3.  判定系统当前所处阶段：稳态运行、失衡加速、临近临界
  4.  标注隐性代价与次生风险
- **单层输出**：核心断层线清单（按严重度排序） + 系统风险预警 + 矛盾本质总结

### 调用模式
1.  **全流程调用**：输入触发口令「调用断层线全链路分析」，自动执行全部7层分析，输出完整报告
2.  **模块化调用**：指定分析层级，例如「调用断层线分析 制度结构层」，仅执行对应模块
3.  **专项场景调用**：保留原有专项触发口令，自动匹配对应模块组合：
    - 调用结构性无辜分析
    - 调用自愿强迫劳动分析
    - 调用防呆治理悖论分析

### 输出规范
统一遵循「事实基准→结构拆解→核心断层→风险判定」四段式结构，无抒情铺垫，所有结论附带推导依据，关键数据标注可信度评级。
